**任务路线**：vm+linux(2)->搭建Hadoop集群(3)->HDFS初步(2)->MR初步(3)->Hive初步(2)->Spark初步(3)->实践(3)

**定位人群**：Linux基础/Shell/Python

**难度系数**：中

**每个任务完成大概所需时间**：3-4h
